## Visão Geral

<complexity>MEDIUM</complexity>

Escolher e configurar ferramenta de load testing (k6 ou Artillery), criar scripts de teste para endpoints principais (login, projetos, etc), definir thresholds de performance (tempo de resposta < 500ms, taxa de erro < 1%), configurar cenários de teste (usuários simultâneos, ramp-up), executar testes de sobrecarga em ambiente de desenvolvimento e documentar resultados com recomendações.

<requirements>
- Ferramenta de load testing configurada (k6 ou Artillery)
- Scripts de teste para endpoints principais (auth, projects)
- Thresholds de performance definidos (tempo de resposta, taxa de erro)
- Cenários de teste configurados (ramp-up, usuários simultâneos)
- Testes executados e resultados documentados
- Relatório de performance com recomendações
</requirements>
